# Feature

A new user-facing capability.

## Grooming

- Ask the user what outcome they want **after this feature ships** — the user-visible change, not the implementation.
- If the user request doesn't already state a clear goal, challenge the user to articulate it. No goal → no `ready-for-dev`.
- Capture the goal in the plan's `business_goal:` frontmatter and the **Business goal** section.
- If the original user request file doesn't mention the goal, update it so the request reflects the agreed outcome.
